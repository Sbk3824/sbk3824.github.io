Image Descriptor using NLP

Ongoing Master’s Project. Building a Model that combines computer vision with natural language processing to devise a methodology to describe an image in natural language, within the context of the question.

The answers may be one word or a detailed description of the image and the output can be plain text or audio depending upon the user requirement. We are using the Visual Genome dataset to train our model. It contains over 108K images and each image can be a collection of multiple attributes and relations.

  • Date: Sept 2018 - Present
  • Category: Data Science

URL Shortening Site

Scalable Bitly like URL Shortening Site

Developed highly scalable and distributed URL shortening service using MERN stack with caching mechanism on link redirection server, messaging queue implementation for better handling of newly shortened links and trending information service with trend database for analytics. Hosted all the servers on AWS EC2 and frontend on Heroku. Able to get 25% performance improvement after implementing caching with Redis and Messaging Queue architecture. Acclaimed as the most efficient project in a class of 80 in an in-session Hackathon.

  • Date: December 2017
  • Category: Photography
  • Technologies Used: JavaScript, MERN, Docker, AWS, Heroku
